== Description ==

Display your logos with their title and description in gallery or slider

You can create custom post type for sparky logos and assign categories. You can simply add the shortcode from post or page editor via clicking the gallery icon.

== Installation ==

Install the plugin into the `/wp-content/plugins/` folder, and activate it.

Create Sparky Logos posts, you can also assign categories.

Insert shortcode via the post editor.
